### 3.2. switch

注意事项：
- switch 条件表达式类型只能是是 btye, short, int, char 和**对应的包装类？**，**JDK 5 新增枚举，JDK 7 新增 String，**
- **case 值必须是字面量**
- case 后的语句和 break 语句可省略。几个 **case 公用一个语句，利用 break 穿透**可以简化代码。

```
switch(literal){
  case: 1
  	sout(..)
  	break;
  case: 2
  case: 3
    sout(..)
  	break;
  default:
    sout(..)
  	break;
}
```
